Chapter 9 Chat

9.1                    Private chat

Users can send instant messages to each other on the private PadLockSL network. The chat dialog is shown in Figure 9.1.1. It can be invoked by double clicking the nickname of the user who we want to chat in the user list of Main GUI. PadLockSL will encrypt the messages before sending them and decrypt messages after receiving them.

Figure 9.1.1 PadLockSL Private Chat Dialog

9.2                    Group chat

User can create or join a chat room and make group chat. The group chat dialog can be invoked by clicking the chat button or use the menu command item in the Main GUI. Also users can use hot key "Alt +J"  to show the dialog. As Figure 9.2.1 shows, a chat room name dialog will be displayed first. User can choose an existent chat room or create a new chat room name. Using a name begin with # or & will create or join a group chat room. The chat room dialogs shows as Figure 9.2.2.

All the members of the chat room are listed in the chat dialog. When a new one join the chat room, all other members of the chat room are informed by a message such as "Tom has joined chat room1". If someone leaves the chat room, all other members of the chat room are informed by a message such as "Tom has left #chat room1". All the messages in the chat room are secured. In the group chat window as shown in Figure 9.2.2, there are three functions available. User can choose a particular person and press the "smile face" button to start a private chat, browse and send file to him/her by using the other two buttons.
