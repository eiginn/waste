Chapter 8 Browse

PadLockSL provides Browse dialog to browse a virtual director structure for each user on the network.

8.1      Invoke the dialog

The Browse dialog can be invoked by click the  button or use the menu command item in the Main GUI dialog. Also users can use hot key "Alt + B" to show the dialog.

8.2      Browse the virtual directory structure

Each user can specify a list of directories to be made available for other users over the network. The directories can be configured during the profile setup or by using the PadLockSL Preferences dialog. As Figure 8.2.1 shows, there are several buttons for different commands. The  button displays the virtual directory structure's root, ie, the user list. You can double click an item in the list view to enter in the related directory. The  button displays the parent directory. The dialog provides a filter facility which can filter the browse content by the text you input in the filter edit line.

Figure 8.2.1 PadLockSL Browse Dialog

8.3      Download files

Users can download selected files in the browse dialog by double clicking the file or by clicking the right key of the mouse and select the "Download" command of the pop menu. The download file's information will be added to the file transfer dialog and can be managed by the file transfer dialog.
