Chapter 5 Setup Connections

5.1                    Copy public key to the clipboard

Users can use PadLockSL to setup private network. The public key can be obtained by using the "Copy my public key to the clipboard " command in the PadLockSL Preferences dialog as Figure 5.1.1 shows. Then users can paste their public key out and send the key to other ones they want to connect to.

Figure 5.1.1 PadLockSL Preference Setting - Authentication

Before setup connection between two users, they should exchange their public keys outside of PadLockSL, such as email or file sharing.

5.2                    Add others' public key

If users get others' public keys, they must add those public keys to their PadLockSL configuration by using the PadLockSL Preferences Dialog as Figure 5.2.1 shows. When users press "Add", a dialog will pop up as Figure 5.2.2 shows. Users can import public key from files and press "OK" or paste public key text to the edit area of the dialog and press "Load key text".

Figure 5.2.1 PadLockSL Preference Setting - Trusted Friends

Figure 5.2.2 PadLockSL Import Public Key Dialog

5.3                    Connect to host

If both users have got each other's public keys, they can type the peer user's IP address in the network status dialog as Figure 5.3.1 shows, and connect each other to be a private network. For example, person A wants to connect private network with Person B (IP Address: 192.168.16.71). Person A should type 192.168.16.71 in the network status window shown as below and press the connection button to connect to Person B.

Figure 5.3.1 PadLockSL Network Status

If a third user wants to join the existing private network, he/she must exchange his/her public key with at lease one user of the private network first, then the connection can be setup just as the same way as two users do.
