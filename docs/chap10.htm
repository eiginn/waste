Chapter 10 File Transfer

User can send files or download files from other users. Files can be found via the file browsing feature, or files can be sent to other users manually.

10.1              Invoke the dialog

The file transfer dialog can be invoked by the menu command item in the Main GUI dialog. Also users can use hot key "Alt + T" to show the dialog.

10.2              Send files

Files can be sent to other users manually by clicking the right key of the mouse and use "send files to user" command of the pop menu in the Main GUI. The sending files' information will be added to the file transfer dialog and can be managed by the file transfer dialog.

10.3              File transfer dialog

The download/upload files can be managed with the PadLockSL File Transfer dialog as Figure 10.3.1 shows. File Transfer dialog shows the status of downloading or uploading including the information about filename, size, location and the transfer speed. Buttons are provided to enable users to abort, resume, remove or clear the downloads/uploads

 If a sender aborts an upload of a file which is not in the sharing path, the receiver can not resume the related download.
