Chapter 11 Preference

Users can use preference dialog to configure PadLockSL in detail.

11.1              Invoke the dialog

The preference dialog can be invoked by click the preference button or use the menu command in the Main GUI dialog. Also users can use hot key "Ctrl + P" to show the dialog.

11.2              Preference dialog

As Figure 11.2.1 shows, the Preference dialog provides several pages for different configuration items including General, Authentication, Trusted friends, and File transfer. Users can choose the pages by clicking the corresponding item name in the tree view area of the Preference dialog.

Figure 11.2.1 PadLockSL Preference Dialog - General

11.3              General

As Figure 11.2.1 shows, users can configure the nickname (display name when setup) and real name/user info (full name when setup) in this page

  

11.4              Authentication

As Figure 11.4.1 shows, users can configure the private key information including create new private/public key pair, copy public key to the clipboard, choose the AES key size, and etc.

Figure 11.4.1 PadLockSL Preference Dialog - Authentication Page

11.5              Trusted Friends

As Figure 11.5.1 shows, users can add or remove public keys, and view the keys' text and etc. How to add public keys is described in Chapter 5.2.

Figure 11.5.1 PadLockSL Preference Dialog - Trusted friends Page

11.6              File Transfer

As Figure 11.6.1shows, users can configure the options about receiving including the path to save downloaded files and etc.

 

 If users change the scan directories, they should press the "Refresh" button
