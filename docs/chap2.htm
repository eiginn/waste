Chapter 2 Getting Started

2.1                    System Requirement

2.1.1    Software

Operating System: RedHat Linux 9.0 or Microsoft Windows XP/2000.

PadLockSL Software: PadLockSL softwaresetup package for Linux or PadLockSL setup program for Windows..

2.1.2    Hardware

Personal computer with or without VIA C5P processor.

2.2                    Software Package Name Form

2.2.1    Setup package for Linux

The name form of the setup package for Linux is PadLockSL.AA.BB.YYMMDD.tgz. PadLockSL is the software name. AA.BB.YYMMDD is the setup package version information. AA is the major version number, and BB is the minor version number. YYMMDD is the time information, which YY for year, MM for month, and DD for day. For example, PadLockSL.01.02.040303 means major version number is 01, minor version number is 02, and building time is 2004-03-03.

2.2.2    Setup program for Windows

The name form of setup program for Windows is PadLockSL.win.AA.BB.YYMMDD.exe. PadLockSL is the software name. Win means the software for Windows. AA.BB.YYMMDD is the software package version information. AA is the major version number, and BB is the minor version number. YYMMDD is the time information, which YY for year, MM for month, and DD for day. For example, PadLockSL.win.01.01.040303 means program for Windows, major version number is 01, minor version number is 02, and building time is 2004-03-03.

2.3                    Installation for Linux

PadLockSL software operates in Linux X-Window. PadLockSL package needs to be installed before running. If users have the setup package for Linux, the package provides shell script files to install/uninstall PadLockSL.

2.3.1    Install the package

Copy the setup package file to the RedHat linux file system such as /home/Test_PadLockSL, then extract the package to generate directory PadLockSLYYMMDD.

Linux# cd /home/Test_PadLockSL

Linux# tar zxvf PadLockSL.AA.BB.YYMMDD.tgz

Enter the directory PadLockSLYYMMDD, execute the shell script file install.sh to install the corresponding files to system directory: related library files are copied to /usr/local/lib, a new directory /usr/local/PadLockSL will be created.

Linux# cd PadLockSLYYMMDD

Linux# ./install.sh

 

 

 

2.3.2    Uninstall the package

Enter the directory PadLockSLYYMMDD, execute the shell script file uninstall.sh to remove the corresponding library files from system directory and remove the /usr/local/padlocksl directory.

Linux# cd PadLockSLYYMMDD

Linux# ./uninstall.sh

2.4                    Installation for Windows

PadLockSL software operates in Microsoft Windows XP/2000. PadLockSL package needs to be installed before running. If users have the setup program for Windows, the program is a shield executable program to install/uninstall.

2.4.1    Install the program

Copy the PadLockSL setup program to the Windows XP/2000 file system such as D:\test\, then execute the setup program. The setup wizard will guide you to install PadLockSL to the destination path you prefer to, copy related lib files to system path, add a shortcut of PadLockSL in the desktop, and add a folder of PadLockSL to the programs group of start menu.

2.4.2    Uninstall the program

Users can use the PadLockSL setup program to remove PadLockSL, or use the Windows Add/Remove Programs tool to remove it.

2.5                    Execution

2.5.1    Execute in Linux

Users can use PadLockSL command to execute PadLockSL.

Linux# padlocksl

2.5.2    Execute in Windows

Users can execute PadLockSL through the shortcut of PadLockSL in the desktop or the start menu.
