Chapter 7 Network status

PadLockSL provides network status dialog to manage network connections.

7.1                    Invoke the dialog

The network status dialog can be invoked by clicking the  button or use the menu command item in the Main GUI dialog. Also users can use hot key "Alt + N" to show the dialog.

7.2                    Manage the connections

Figure 7.2.1 shows, the network status dialog has an edit line. Users can input the host address then press the button which is next to the edit line. If the public keys have been exchanged before, a new connection will be set up. Users can select a connection from the connection list in the dialog then press the buttons to reconnect, disconnect or disconnect and remove.
