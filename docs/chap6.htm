Chapter 6 Basic Operations

6.1                    Network status

PadLockSL provides network status dialog to manage network connections. Users can connect to a host, disconnect from or remove from selected host.

6.2                    Browse

Users can use browser dialog to browse a virtual directory structure for each user on the network. Each user can specify a list of directories to make available to other users on the network. The directories can be configured during the profile setup or by using the PadLockSL Preferences dialog. User can directly download files or directories from the browser dialog.

6.3                    Chat

6.3.1    Private chat

Users can send instant messages to each other on the private PadLockSL network.

6.3.2    Group chat

User can create or join a chat room and make group chat.

6.4                    File Transfer

User can send files to or download files from other users. Files can be found via the file browsing feature, or files can be sent to other users manually. The download/upload files can be managed with the PadLockSL File Transfer window

6.5                    Preference

Users can use preference dialog to configure PadLockSL in detail. The Preference dialog provides several pages for different configure items including identity, network, file transfer.
