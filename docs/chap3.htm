Chapter 3 Profile setup

A Profile Setup Wizard Window will display when first running PadLockSL. The Profile setup includes three steps. You can configure the basic information of PadLockSL step by step by following the Setup Wizard.

3.1                    Step one

In this step, you can configure the identity information. As Figure 3.1.1 shows, which includes your display name, and User information. It is set as "PadLock SL User" for User information by default, user can change to another name.

Figure 3.1.1 PadLockSL Profile Wizard’ General Information

3.2                    Step two

In this step, you can configure the network security information. As Figure 3.2.1 shows, you may choose to create public/private key pair or to import an old one. If you choose to create public/private key pair, press the button “create key pair”, and a PadLockSL Key Generator Dialog will pop up as shown in Figure 3.2.2 shows. You should enter the password you want to use in the Passphrase edit line. Then you can choose a preferred Key size and press the button “Generate”. The dialog will change to a key generating status bar as Figure 3.2.3 shows. When the key generation is accomplished, there will be a message dialog as in Figure 3.2.4 to indicate the key generation is done. Then you can press “OK” and “Next” to the next step.

Figure 3.2.1 PadLockSL Profile Wizard’ Network Security

Figure 3.2.2 PadLockSL Key Generator

Figure 3.2.3 PadLockSL Key Generator Status Bar

Figure 3.2.4 PadLockSL Key Generator’ Completion Confirmation

3.3                    Step three

In this step, you can configure the file transfer information. As Figure 3.3.1 shows, you may set the path to receive files, and the path you want to share files with other users. Sharing path can make files on local computer accessible to the private network. When a user uses file browsing service to browsing the private network, he/she can access the files in the sharing path and download them. When you have done the entire configuration, you can press the “Done” to run PadLockSL.

Figure 3.3.1 PadLockSL Profile Wizard’ File Transfer

The Profile Setup Wizard of PadLockSL provides enough configuration information for PadLockSL to setup connection with others. If user wants to configure more options about PadLockSL, he can use the Preference dialog.
