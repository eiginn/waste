Chapter 4 Login

4.1                    Input password

PadLockSL uses password to provide basic authentication. As Figure 4.1.1 shows, users must input correct password to load their private keys to setup the PadLockSL private network with others. If users input the wrong password, an error message will pop up as in Figure 4.1.2. If users input correct password and press "OK" , the Main GUI will show. If users press the "Cancel" button, he/she can only import a private key. That is, the private key is encrypted by the password, so if you bypass the passphrase, PadLockSL allows users to use GUI, but the user will not be able to communicate with others until he/she imports a valid private key.
